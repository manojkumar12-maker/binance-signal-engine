🚀 Deploy to GitHub Pages

Step 1: Enable GitHub Pages

  1. Go to your repo: https://github.com/manojkumar12-maker/binance-signal-engine
  2. Go to Settings → Pages
  3. Under Source, select Deploy from a branch
  4. Select main branch and / (root) folder
  5. Click Save

Step 2: Wait for Deployment

Your site will be live at:

https://manojkumar12-maker.github.io/binance-signal-engine/

Step 3: Update Backend URL

Edit frontend/index.html line ~170 and update the WebSocket URL to your server:

const wsUrl = 'wss://your-server.com/ws';
