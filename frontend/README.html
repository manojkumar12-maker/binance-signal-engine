🚀 Binance Signal Engine — Deployment

Backend: Railway

  1. Go to railway.app and create new project
  2. Connect GitHub repo: manojkumar12-maker/binance-signal-engine
  3. Add PostgreSQL plugin (database)
  4. Add Redis plugin (cache)
  5. Railway auto-sets DATABASE_URL, REDIS_URL, and PORT
  6. Add Telegram env vars if needed

Frontend: GitHub Pages

  1. Go to repo Settings → Pages
  2. Under Source, select GitHub Actions
  3. Push to main branch to trigger deploy

Your frontend will be live at:

https://manojkumar12-maker.github.io/binance-signal-engine/

Update Backend URL

Edit frontend/index.html and set your Railway URL:

window.BACKEND_URL = 'https://your-app.up.railway.app';
